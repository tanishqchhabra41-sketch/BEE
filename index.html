Introduction
A transformer is a static electrical device that transfers electrical energy from one circuit to another using electromagnetic induction.
Slide 2
Transformer on No Load
Secondary winding is open.
Only no-load current flows in primary.

I₀ = Iₘ + Iw
Slide 3
No Load Losses
• Hysteresis Loss
• Eddy Current Loss

Copper loss is negligible.
Slide 4
Transformer on Load
Load is connected to secondary.
Current flows in both windings.

Power is transferred to load.
Slide 5
Current Relation
I₁ / I₂ = N₂ / N₁
Slide 6
Losses on Load
Total Loss = Core Loss + Copper Loss

Copper loss increases with load.
Slide 7
Auto Transformer
Uses a single winding.
Part of winding is common.

More efficient and economical.
Slide 8
Copper Saving
Copper Saving = (1 − V₂/V₁) × 100%
Slide 9
Conclusion
• No load → only core losses
• Load → power transfer
• Auto transformer → saves copper
